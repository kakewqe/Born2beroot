# Born2beroot

## 🇬🇧 English

The goal of this project is to set up a secure and functional Linux server.

### Description

`Born2beroot` introduces system administration and virtualization.

You will create a virtual machine and configure it following strict security rules.

This project helps you better understand:

* Linux system administration
* virtualization
* partitioning and LVM
* user and group management
* SSH configuration
* firewall and security policies

### Features

* Secure SSH service
* Configured firewall (UFW)
* Strong password policy
* User and group management
* Sudo configuration with rules
* Monitoring script (system information display)

### Tools & Technologies

* Virtual machine (VirtualBox or similar)
* Debian or Rocky Linux
* SSH
* UFW (firewall)
* Bash scripting

### Skills Developed

* system administration
* Linux command line
* security best practices
* scripting
* networking basics

## 🇫🇷 Français

Le but de ce projet est de mettre en place un serveur Linux sécurisé et fonctionnel.

### Description

`Born2beroot` est une introduction à l'administration système et à la virtualisation.

Vous devez créer une machine virtuelle et la configurer en respectant des règles strictes de sécurité.

Ce projet permet de mieux comprendre :

* l'administration système Linux
* la virtualisation
* le partitionnement et LVM
* la gestion des utilisateurs et groupes
* la configuration de SSH
* les règles de sécurité et pare-feu

### Fonctionnalités

* Service SSH sécurisé
* Pare-feu configuré (UFW)
* Politique de mot de passe stricte
* Gestion des utilisateurs et groupes
* Configuration de sudo avec règles
* Script de monitoring (affichage des informations système)

### Outils & Technologies

* Machine virtuelle (VirtualBox ou équivalent)
* Debian ou Rocky Linux
* SSH
* UFW (pare-feu)
* Bash

### Compétences développées

* administration système
* ligne de commande Linux
* bonnes pratiques de sécurité
* scripting
* bases réseau
